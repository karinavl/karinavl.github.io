## Site Karina Vieira

### Theme

> https://rubygems.org/gems/freelancer-theme-jekyll

![Jekyll logo](https://jekyllrb.com/img/logo-2x.png)

### Install

> https://jekyllrb.com/docs/installation/ubuntu/

### Install gems

```sh
bundle install
```

### Jekyll commands

- Build

    ```
    bundle exec jekyll build
    ```

- Local test server

    ```sh
    bundle exec jekyll serve
    ```
